<h1>QuickDrop</h1>
<h3>Fastest way to transfer data</h3>






Install The Latest Version of QuickDrop 






Give Permission for All File Access





Connect Other Device with same Wifi or Phone's Hotspot (5Ghz Recommended for fastest Speed)







<h2>Share the entire Phone Storage</h2>





Open the App





Scan the QR Code from other device or type shown ip address on browser





Now You can view the entire phone storage on the browser and download anything by clicking the file





<h2>Share Selected files</h2>





Select any files click Share and select the app QuickDrop





Scan the QR Code from other device or type shown ip address on browser





Now You can download shared files





(Uncheck Private Sharing if you also want to view phone's storage)
